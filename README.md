# 📉 Customer Churn Intelligence Platform  
### End-to-End Machine Learning & Cloud Analytics Solution

Customer Churn Intelligence Platform is a full-stack data analytics project designed to predict customer churn and generate actionable retention insights using Machine Learning, Cloud Databases, and Business Intelligence tools.

This project simulates a real-world enterprise analytics workflow — transforming raw telecom customer data into predictive intelligence and executive dashboards.

---

## 🧠 Project Overview

Customer churn is one of the most critical business challenges in subscription-based industries. This project demonstrates how modern data teams design scalable churn prediction systems by integrating:

- Data preprocessing & feature engineering
- Machine learning model development
- Churn probability scoring
- Cloud-based database integration
- Interactive business dashboards

The system identifies high-risk customers and supports data-driven retention strategies.

---

## 🎯 Key Objectives

- Predict whether a customer is likely to churn
- Generate churn probability scores
- Identify high-risk customer segments
- Analyze churn behavior by:
  - Contract Type
  - Payment Method
  - Tenure
  - Monthly Charges
- Build a scalable cloud-integrated analytics pipeline
- Deliver executive-ready dashboards for business teams

## 🔄 End-to-End Data Pipeline Architecture

1️⃣ Load raw telecom dataset (CSV format)  
2️⃣ Perform data cleaning & preprocessing using Python  
3️⃣ Encode categorical variables & engineer new features  
4️⃣ Train Machine Learning classification model  
5️⃣ Generate churn predictions & probability scores  
6️⃣ Export processed dataset with predictions  
7️⃣ Upload dataset to Azure SQL Database  
8️⃣ Connect Azure SQL to Power BI  
9️⃣ Develop interactive dashboards using DAX measures  

This pipeline mirrors a production-level analytics workflow.
